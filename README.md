# VisionX-AI
AI-powered system combining virtual mouse, real-time object detection, and face recognition.

# VisionX AI

> Transforming the way humans interact with machines using Computer Vision

---

## Overview
VisionX AI is an advanced computer vision system that enables users to interact with their computer in a smarter and more intuitive way.  
It combines multiple AI modules into one unified project:

- Virtual Mouse (gesture-based control)
- Object Detection (real-time recognition)
- Face Recognition (identity detection)

---

## Features
- Control cursor using hand gestures
- Click, scroll, and navigate without a physical mouse
- Detect and classify objects in real-time
- Recognize and identify human faces
- Fast and efficient performance using optimized models
